# DataAnalyisis_Task2

# Predictive Modeling Using Machine Learning

## Project Overview
This project focuses on building a machine learning model to predict product availability based on product attributes such as brand, category, price, stock quantity, color, and size.

## Objective
Develop a supervised learning model capable of predicting product availability and evaluate its performance using standard machine learning metrics.

## Dataset
The dataset contains product-related information including:
- Brand
- Category
- Price
- Stock
- Color
- Size
- Availability

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Machine Learning Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Prediction
7. Model Evaluation

## Algorithm Used
- Random Forest Classifier

## Evaluation Metrics
- Accuracy Score
- Classification Report
- Confusion Matrix

## Results
The model was trained and tested on the dataset to predict product availability. Performance was evaluated using classification metrics and visualizations.
